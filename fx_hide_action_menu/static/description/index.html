Hide Action Menu

OVERVIEW

This module hide the Action button for the particular user in Odoo vesion 17.

SCREENSHOTS

Action Menu in List view


Action menu is viewing in list view, when it is not hidden.

Action Menu in Form view


Action menu is viewing in form view, when it is not hidden.

Configuration User: Settings > User & Companies > Users


Admin can hide action menu from all applications by selecting the field "Hide Action All", and if admin want to hide action menu for selecting applications then, admin can select application in field "Applied Models for Hiding Action".

Action menu hidden in List view


Action menu is hidden in list view, which is configured in settings.

Action menu hidden in Form view


Action menu is hidden in Form view, which is configured in settings.
